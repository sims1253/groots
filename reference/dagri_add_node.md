# Add a node to a dagriculture graph

Add a node to a dagriculture graph

## Usage

``` r
dagri_add_node(
  graph,
  id,
  kind,
  label = NULL,
  params = list(),
  metadata = list()
)
```

## Arguments

- graph:

  A `dagri_graph`.

- id:

  Node ID.

- kind:

  Node kind.

- label:

  Node label.

- params:

  Node parameters.

- metadata:

  Node metadata.
