# Update a node in a dagriculture graph

Update a node in a dagriculture graph

## Usage

``` r
dagri_update_node(graph, node_id, label = NULL, params = NULL, metadata = NULL)
```

## Arguments

- graph:

  A `dagri_graph`.

- node_id:

  Node ID.

- label:

  Node label.

- params:

  Node parameters.

- metadata:

  Node metadata.
